📺 Prime Video Clone

A fully responsive, modern UI clone of Amazon Prime Video — built only for learning, UI practice, and portfolio demonstration.
This project does not promote piracy or redistribute any copyrighted content. It is simply a frontend replica created for skill development.

📝 About the Project

This project is a frontend-only recreation of the Prime Video interface.
It replicates the design, layout, and user experience using HTML, CSS, and JavaScript.

The goal of this project was to practice:

Modern responsive web design
Layout structuring
Styling with clean and organized CSS
DOM manipulation
Carousel & slider effects
UI/UX design thinking

This clone does not fetch, store, or stream any copyrighted content.
All images, titles, and posters used are sample/demo assets for educational use.

🚀 Features
🎨 Fully responsive UI
🖼️ Movie & show card layouts
🎬 Hero banner section
🎞️ Category-based carousels
🌙 Dark theme similar to Prime Video
🧭 Functional navigation UI
🔄 Smooth transitions & hover animations
🎯 Clean code structure (HTML / CSS / JS separated)
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling, layouts, responsiveness
JavaScript – Sliders, interactions, UI logic
